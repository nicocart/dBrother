项目简介
dBrother 是一个专业的 PDF 孔径分析工具，围绕实验报告实现全结构化抽取与可视化展示。最新版利用 pdfplumber 的表格定位能力直接读取原始数据，辅以多段插值与异常检测，确保分析结果更稳定、可追溯。

开源依赖
本项目基于以下优秀的开源项目构建：

FastAPI - 现代、快速的Web框架
pdfplumber - PDF 表格/文本解析库
Bootstrap - 响应式UI框架
Chart.js - 数据可视化图表库
jQuery - JavaScript工具库
技术特性
基于 pdfplumber 的表格/字段结构化解析
多指标自动计算（D10/D90、孔容分段等）
交互式图表与原始数据导出
响应式布局，支持移动端体验
解析过程校验与错误提示
隐私保护：文件处理后立即清理
